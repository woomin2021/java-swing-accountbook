# Java Swing 가계부 프로그램

Java Swing을 활용하여 제작한 데스크톱 기반 가계부 관리 프로그램입니다.

## 주요 기능
- 수입 / 지출 등록
- 날짜별 내역 조회
- 카테고리별 지출 관리
- 총 수입 · 총 지출 · 잔액 자동 계산
- MySQL 연동을 통한 데이터 영속성 관리

## 🛠 사용 기술
- Java (JDK 8+)
- Java Swing (GUI)
- JDBC
- MySQL
- IntelliJ IDEA

## 📂 프로젝트 구조
- model : 데이터 모델
- dao : DB 접근 로직
- ui : Swing 화면 구성
- Main : 프로그램 실행 진입점



※ DB 접속 정보는 보안상 제외되어 있습니다.
